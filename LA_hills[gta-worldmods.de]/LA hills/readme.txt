Thanks for downloading my mod...

This was my first small map-mod I made for gta in the mid of 2007. Because of beeing quite unhappy with the style of the Mulholland safehouse (not looking worth its price) I decided to change the textures and to add some new objects.
Now, after I installt the "real palms mod" and the "project oblivion" I changed my old mod and decided to release it.

If you want to install the mod, it is strongly recommended to install the "San Andreas-Real palm's v2.0" that can be found here:

http://gta-worldmods.t-n-network.de/phpkit/start/include.php?path=content/overview.php&type=4&contentid=2729

and the "New Project Oblivion" mod that can be found here:

http://gta-worldmods.t-n-network.de/phpkit/start/include.php?path=content/overview.php&letter=&catid=75&themeid=&type=4&order=&entries=60

At least you should install "San Andreas-Real palm's v2.0"!! (if not, there is a map-file in my "without real palms mod" folder for these cases).

How to install:

- open the gta3.img with an img-tool like spark, Alci's IMG Editor 1.5 or IMG Tool
- replace the files "mullho03_lahills" and "lahills_safe1.txd" (choose between orange or black/white) with the 2 
files of my "textures"folder  

- rebuild archive

- copy/paste "LAhills" (in my "map" folder)into \data\maps\LA  
	(without the real palm mod use the file of my "without real palms 	mod" folder)

- if you want to have some cars (Windsor in orange, Stafford in black and Cheetah in red) to be parked, use the "carplace_hillhouse.cs" by copying it into your cleo-folder.

Note: If you want to change the cars that are placed with cleo, you can do this very easy: just have a look at my small tutorial.


Deutsch:

Für die Map solltet ihr unbedingt "San Andreas-Real palm's v2.0" installiert haben, idealerweise auch "New Project Oblivion" (links siehe oben). Ohne diese beiden mods schaut es sonst deutlich schlechter aus.
Falls ihr diese beiden mods aber nicht installieren wollt/könnt, nehmt die Datei aus dem "without real palms mod"-Ordner.  

Installation:
- mit einem img-tool wie spark, Alci's IMG Editor 1.5 oder IMG Tool die Datei gta3.img öffnen
- die Dateien "mullho03_lahills" und "lahills_safe1.txd" (entweder orange oder black/white )mit denen aus meinem "textures"- Ordner ersetzen

- "rebuild archive" mit dem IMG-Tool durchführen

- per copy/paste die Datei "LAhills" (aus meinem map-Ordner) in den Ordner \data\maps\LA einfügen 

- falls ihr noch drei Wagen dort oben platzieren möchtet, einfach die Datei "carplace_hillhouse.cs" in euren Cleo-Ordner einfügen (natürlich muss dazu Cleo installiert sein ;) )

Wenn ihr die Wagen durch andere ersetzen möchtet, einfach dem Tutorial folgen.
